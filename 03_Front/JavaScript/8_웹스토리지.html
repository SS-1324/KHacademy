웹 스토리지

브라우저에 데이터를 key-value로 저장하는 공간
- localStorage : 영구저장(브라우저를 닫아도 유지)
- sessionStorage : 탭 단위 저장(탭을 닫으면 사라짐)
메서드 : setItem(key, value), getItem(key), removeItem(key), clear()
웹스토리지는 '문자열'만 저장이 가능
